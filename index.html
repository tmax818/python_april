The Four Ways of Interpolation

File Syntax Notes
server.py 
            @app.route('route/<int:id>')
            def route(id):
              return render_template('index.html', id = id)
<int:id> used to insert variables in routes id must be passed down to the python function. This syntax is ONLY available in route decorators!
*.html 
            {{id}}
jinja tags {{id}} used to render content in any html page. id must be passed from the render_template function.
*.py inside python functions 
            f"{id}"
Python code requires Python syntax. Inside Python functions, use string interpolation where necessary
